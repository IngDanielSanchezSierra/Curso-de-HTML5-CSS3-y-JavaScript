### <a name="2-1"></a>2.1 De HTML básico a la Web Moderna con HTML5, CSS3 y JavaScript

Hoy vamos a hacer un pequeño viaje en el tiempo y ver cómo todo esto comenzó.

![Fig. 1 - Ejemplo de una página web en HTML](media/Fig.%201%20-%20Ejemplo%20de%20una%20página%20web%20en%20HTML.png)

*Fig. 1: Ejemplo de una página web en HTML.*

---

Imagina la web en sus primeros días, cuando apenas existía una versión muy básica de HTML.  
El propósito era sencillo: dar una estructura básica al texto para que los documentos pudieran compartirse entre computadoras a distancia.  
Era como si solo quisiéramos decirle a la computadora:  

> "Esto es un título, esto es un párrafo".  

Nada de colores, nada de imágenes… era solo texto estructurado.

---

![Fig. 2 - Ejemplo de una página web en HTML y CSS](media/Fig.%202%20-%20Ejemplo%20de%20una%20página%20web%20en%20HTML%20y%20CSS.png)

*Fig. 2 - Ejemplo de una página web en HTML y CSS.*

Pero claro, la tecnología no se queda quieta.  
Con el tiempo, mejoraron las pantallas, llegó el color, y de repente queríamos más:  

- Imágenes  
- Diferentes tipos de letra  
- Diseño más atractivo  

En ese momento, el HTML básico ya no era suficiente y a los desarrolladores se les empezó a complicar la cosa.  
Ahí es cuando entra en escena **CSS**, que permitió separar el contenido (**HTML**) de la presentación (**CSS**).  

> HTML estructura, CSS diseña.  

Era un respiro enorme para los desarrolladores, que ya no tenían que romperse la cabeza tratando de hacer que el diseño y el contenido funcionaran juntos de manera rígida.

---

Pero aún había algo que faltaba.  
HTML y CSS nos daban un sitio bonito, pero **no podíamos interactuar realmente con él**.

![Fig. 3 - Ejemplo de una página web en HTML, CSS y JavaScript](media/Fig.%203%20-%20Ejemplo%20de%20una%20página%20web%20en%20HTML%2C%20CSS%20y%20JavaScript.gif)

*Fig. 3 - Ejemplo de una página web en HTML, CSS y JavaScript.*

Fue entonces cuando **JavaScript**, que ya estaba por ahí, dio un paso al frente.  
Al estar integrado en los navegadores, JavaScript podía:

- Acceder al contenido HTML de la página  
- Manipularlo en tiempo real  
- Pedir datos al servidor sin recargar la página  
- Procesar información y mostrar resultados  

¡En resumen, convertía la web en algo dinámico!

---

Así que ya teníamos los tres grandes:  

- **HTML** → estructura  
- **CSS** → diseño  
- **JavaScript** → vida e interacción  

Pero había un problema: los tres lenguajes habían evolucionado por caminos separados, generando algunas inconsistencias.  
La solución vino con **HTML5**, una versión que unificó todo.  

> HTML estructura, CSS diseña, y JavaScript le da vida.  
> ¡La web como la conocemos hoy estaba lista!

---

Ahora, en este curso, vamos a sumergirnos en cómo usar cada uno de estos lenguajes para crear sitios y aplicaciones web que:  

- No solo se vean bien  
- Sino que también funcionen de maravilla
