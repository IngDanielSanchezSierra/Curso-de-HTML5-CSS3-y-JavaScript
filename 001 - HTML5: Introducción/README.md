## ¿Qué es HTML y su versión moderna HTML5?

HTML significa **HyperText Markup Language** (Lenguaje de Marcado de Hipertexto). HTML5 es la quinta y más reciente versión de este lenguaje.

HTML no es un lenguaje de programación, ya que no permite crear lógica o realizar cálculos. Su función es definir la estructura y organizar el contenido de una página web. :contentReference[oaicite:0]{index=0}

HTML5 introdujo mejoras importantes, como:

- Soporte para audio y video.
- Formularios más avanzados.
- Mejor organización del contenido.
- Nuevas etiquetas y funcionalidades modernas. :contentReference[oaicite:1]{index=1}

## Estructura básica de un documento HTML

```html
<!DOCTYPE html>
<html lang="es">
    <head>
        <meta charset="UTF-8">
        <title>Título de la página</title>
    </head>

    <body>
        <span>¡Bienvenid@!</span><br>
        <span>Esta es una página HTML muy simple.</span>
    </body>
</html>
