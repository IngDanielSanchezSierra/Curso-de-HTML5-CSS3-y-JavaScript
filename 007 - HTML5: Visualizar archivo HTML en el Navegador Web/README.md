## 📖 Contenido del capítulo

Antes de definir los elementos de la cabecera <head>, vamos a visualizar los cambios que realizamos en nuestro archivo HTML dentro del navegador. Este es un tema básico, así que si ya conoces el procedimiento, puedes avanzar al siguiente capítulo.

---

### <a name="7-1"></a>7.1 Abrir un archivo HTML usando Live Server

La primera forma que aprenderemos para visualizar nuestros archivos HTML en el navegador es utilizando la extensión Live Server que instalamos en el capítulo donde aprendimos a preparar nuestro entorno de desarrollo.

Para usarla:

1. Haz clic derecho sobre el archivo HTML desde el panel Explorer de VS Code.
2. Y selecciona Open with Live Server.

Esto abrirá tu navegador (en este caso Google Chome) mostrando tu archivo HTML. Como todavía no tiene contenido visible, verás una página en blanco.

La gran ventaja de Live Server es que actualiza la página automáticamente cada vez que guardas cambios en tu archivo HTML. No tienes que presionar F5 ni recargar manualmente.

Para hacer una prueba rápida, escribe temporalmente dentro del <body>: ¡Hola Mundo!

Guarda el archivo presionando la combinación de teclas Ctrl + S, o bien haciendo clic en el menú File y seleccionando la opción Save. Después de guardar, verás el cambio reflejado de inmediato en el navegador.

### <a name="7-2"></a>7.2 Abrir un archivo HTML de forma manual

Si no tienes instalada la extensión Live Server, también puedes abrir tus archivos HTML manualmente.

Para hacerlo:

1. Haz clic derecho sobre el archivo HTML en el panel Explorer.
2. Selecciona Reveal in File Explorer o Revelar en el Explorador de Archivos.
3. En la ventana del Explorador, puedes abrir el archivo dando doble clic sobre él.
   Si al dar doble clic el archivo no se abre en Chrome, puedes hacer clic derecho sobre el archivo HTML, elegir la opción Abrir con, y seleccionar Google Chrome del menú emergente.

De esta forma también puedes visualizar los cambios que vayas haciendo. Sin embargo, esta opción no actualiza automáticamente la página, así que después de guardar los cambios de tu archivo, deberás refrescar la página manualmente:

- Presionando F5 o haciendo clic en el botón de recargar del navegador.

Este es un método totalmente válido, pero Live Server facilita mucho el proceso durante el desarrollo.
