📌 Tu primer archivo HTML
En esta clase aprenderás la relación entre HTML, CSS y JavaScript, por qué es importante separarlos en archivos independientes, y darás tu primer paso práctico creando un archivo index.html en Visual Studio Code.

📖 Relación entre HTML, CSS y JavaScript
Recordemos cómo estas tres tecnologías trabajan juntas para construir páginas web:
🔹 HTML → estructura
🔸 CSS → diseño y estilos
⚡ JavaScript → interactividad

💡 ¿Por qué separar HTML, CSS y JavaScript en archivos diferentes?
Aprende la importancia de mantener tu proyecto organizado:
🔹 HTML → extensión .html
🔸 CSS → extensión .css
⚡ JavaScript → extensión .js

📂 Creación de un archivo HTML en Visual Studio Code
Paso a paso para crear tu primer archivo index.html dentro de una carpeta de proyecto.

💡 ¿Por qué usar index.html?
Conoce por qué este archivo suele ser la página de inicio predeterminada en los servidores web.

✅ Conclusión
Aprendiste la importancia de separar HTML, CSS y JavaScript, además de crear tu primer archivo HTML. ¡Es el inicio de todo proyecto web!
